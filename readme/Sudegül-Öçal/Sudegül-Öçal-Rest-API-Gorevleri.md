# Sudegül Öçal - REST API Görevleri

- **Endpoint Geliştirme:** `/meals` ve `/categories` uç noktaları için gerekli CRUD operasyonlarının (Node.js/Express vb. ile) yazılması.
- **Veritabanı Entegrasyonu:** Yemek ilanları ve kategori modellerinin DB şemalarının oluşturulması.
- **Validasyon Katmanı:** Porsiyon ve fiyat bilgilerinin doğru formatta girildiğinin kontrol edilmesi.
