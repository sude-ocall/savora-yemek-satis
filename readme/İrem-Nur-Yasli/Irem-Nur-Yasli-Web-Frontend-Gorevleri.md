## 1. Üye Olma (Kayıt) Sayfası
* **UI Bileşenleri:**
    * Responsive kayıt formu (desktop ve mobile uyumlu).
    * Şifre güç göstergesi.
    * Başarı/Hata bildirimleri (Toasts).
* **Form Validasyonu:**
    * JavaScript real-time validation.
    * Client-side ve server-side doğrulama eşleşmesi.
* **Teknik Detaylar:**
    * Framework: React/Vue/Angular veya Vanilla JS.
    * SEO optimizasyonu (meta tags).

## 2. Adres Tanımlama Sayfası
* **UI Bileşenleri:**
    * Adres başlığı (Ev/İş), Şehir ve Tam Adres alanları.
    * Mevcut adreslerin liste görünümü.
* **Görev:** Kullanıcının teslimat adresi ekleme ve yönetme arayüzü tasarımı.
