# İrem Nur Yaşlı - Mobil Frontend Görevleri

**Mobile Front-end Demo Videosu:** [Link buraya eklenecek]

## 1. Üye Olma (Kayıt) Ekranı
* **UI Bileşenleri:**
    * Email, Şifre, Ad ve Soyad input alanları.
    * "Kayıt Ol" butonu ve "Zaten hesabınız var mı?" linki.
    * İşlem sırasında Loading indicator.
* **Form Validasyonu:**
    * Şifre güvenlik kuralları (min 8 karakter, büyük/küçük harf).
    * Boş alan kontrolü ve butonun dinamik olarak aktif/pasif olması.
* **Kullanıcı Deneyimi:**
    * Form hatalarının input altında gösterilmesi.
    * Keyboard dismiss işlevi.

## 2. Kullanıcı Profil Görüntüleme Ekranı
* **UI Bileşenleri:**
    * Profil fotoğrafı alanı (placeholder).
    * İsim, Soyisim ve Email bilgileri.
    * "Hesabı Sil" butonu (Kırmızı, alt kısımda).
* **Kullanıcı Deneyimi:**
    * Veri yüklenirken Skeleton screen kullanımı.
    * Pull-to-refresh özelliği.
